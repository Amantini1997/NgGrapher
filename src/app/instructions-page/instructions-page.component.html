Welcome to the Instructions Page!

Grapher.js is a website designed to help students visually learning data structures and algorithms.

Your teacher should have provided you with a configuration file in format JSON. That file should be used in the Animation Page to generate the animation. Once the configuration is loaded and accepted byt the software, the animation will automatically appear in your screen. To understand what each component does, please refer to the next section.

There are 3 main components in front of you:
  • Animation section:
    • Here you get the chance to understand the mechanism of an algorithm or to see a data structure behaves when different methods are called.
  • Code:
    • This part gives you an idea of what line of code corresponds to each operation performed in the animation section. Further comments are provided in the the comment box.
  • Player bar:
    • This component allows you to directly interact with the animation by starting it, stopping and resuming it, and selecting the animation speed.
